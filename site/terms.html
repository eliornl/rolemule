2. About This Software

RoleMule is free, open-source software licensed under the MIT License. The source code is publicly available on GitHub. You are running a self-hosted instance — all your data stays in your own database on the machine where this instance is deployed.

The software provides:

  • Job posting analysis and requirements extraction (including PDF, TXT, and DOCX uploads)
  • Company research and insights
  • Profile-to-job matching and fit assessment
  • AI-generated cover letters and application materials
  • Resume optimization recommendations and on-demand CV optimization
  • Interview preparation guides and timed mock interview sessions
  • Hiring outreach drafts (copy-only — you send messages yourself)
  • Career tools (thank you notes, follow-up emails, salary negotiation coaching, job comparison, and more)
  • Chrome extension for analyzing job postings from pages you visit and matching application forms to your profile
  • Optional command-line interface (rolemule) for the same workflows from a terminal or AI coding agents
